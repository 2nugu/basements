# Figure / Data: Lube 2004 H/D Sweep

## Purpose

Validates that the W2-calibrated friction parameter α=0.55 holds across
multiple column-collapse geometries — defending against single-point
calibration criticism. Reproduces the H/D-dependent behavior reported
in Lube et al. 2004 ("Axisymmetric collapses of granular columns",
J. Fluid Mech. 508).

## Data Source

- File: `lube_hd_sweep.csv`
- Generated by: `research/mpm_rigid_coupling/bench_compare_all.cpp` with
  `REPOSE_HD` environment override; α=0.55 (W2 calibration lock-in).
- Date: 2026-05-15

## Axes

- X: H/D (initial column height / diameter ratio) — range 0.5 to 3.0
- Y_a: `angle_geom_deg` — geometric pile angle = atan(h_peak / r_outer)
- Y_b: `rest_particles` — count of settled particles (sanity check)

## Key Observations

Numerical observations only. Interpretation belongs in paper §6.

| H/D | angle_geom (°) | rest fraction | regime |
|-----|---------------|---------------|--------|
| 0.5  | 37.76 | 196/196 (100%) | frustum-intact |
| 0.71 | 35.06 | 233/245 (95%) | transition |
| 1.0  | 28.13 | 343/343 (100%) | transition |
| 1.5  | 25.84 | 539/539 (100%) | runout-dominated |
| 2.0  | 24.62 | 686/686 (100%) | tall-runout |
| 3.0  | 25.17 | 1029/1029 (100%) | tall-runout |

- For H/D ≤ 0.71, measured angle ∈ [35°, 38°] — consistent with
  literature dry-sand repose range 30°–35° (slight upper-bound bias).
- For H/D ≥ 1.5, measured angle plateaus near 25° — the Lube runout
  regime where the pile is wider than tall and the geometric angle
  underestimates the side-slope angle.
- Transition between regimes occurs near H/D ≈ 0.74, matching Lube 2004
  Fig. 8 transition observation.
- 95–100% particle rest fraction across all cases confirms quasi-static
  pile measurement (no spurious oscillation).

## Reproduction

```bash
cmake --build build --config Release --target bench_compare_all
for hd in 0.5 0.71 1.0 1.5 2.0 3.0; do
    REPOSE_HD=$hd ./build/research/mpm_rigid_coupling/Release/bench_compare_all.exe \
        --method M1 --scenario repose_pile
done
```

Results parsed into `lube_hd_sweep.csv` via shell.
